# 🍎 Fruit Spinner App

A simple Android application developed using Java.

This app provides a dropdown menu that allows users to select a fruit.  
When a fruit is selected, its corresponding image is displayed instantly.

## 📌 Overview

This project demonstrates how to implement a dropdown menu in Android using AutoCompleteTextView and ArrayAdapter.  
It is built using a single activity with a clean and user-friendly interface.

## ✨ Features

• Dropdown menu selection  
• Default selected fruit on launch  
• Dynamic image update based on selection  
• Simple and responsive UI  
• Developed using Java and AppCompatActivity  

## 🍌 Fruits Available

• Banana  
• Apple  
• Mango  

## ⚙️ How It Works

When the application starts, Banana is selected by default.  
Selecting a different fruit from the dropdown automatically updates the displayed image.
