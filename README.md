# 🍎 Fruit Spinner App

A simple Android application implemented in both Java (XML) and Kotlin (Jetpack Compose).

This app provides a dropdown menu that allows users to select a fruit.  
When a fruit is selected, its corresponding image is displayed instantly.

---

## 📌 Project Overview

This project demonstrates two different Android UI approaches:

• Java with XML Layout (AppCompat)  
• Kotlin with Jetpack Compose  

Both implementations provide the same functionality but use different UI systems.

---

## ✨ Features

• Dropdown menu selection  
• Default selected fruit on launch  
• Dynamic image update based on selection  
• Clean and simple UI  
• Implemented using two Android approaches  

---

## 🧑‍💻 Implementations Included

### 1️⃣ Java (XML Based UI)
- Uses AppCompatActivity  
- Uses AutoCompleteTextView  
- Uses ArrayAdapter  
- Traditional Android View system  

### 2️⃣ Kotlin (Jetpack Compose)
- Uses ComponentActivity  
- Uses OutlinedTextField  
- Uses DropdownMenu  
- Built with modern declarative UI  

---

## ⚙️ How to Run

By default, the Java activity is set as launcher in the AndroidManifest file.

If you want to run the Jetpack Compose version:

1. Open AndroidManifest.xml  
2. Change the launcher activity name to:

   .FruitSpinnerKotlin  

3. Run the application
